## Usage
  - clone the repository
    ```
    git clone https://github.com/vamc-stash/Dev-Challenges.git
    ```
  - navigate to respective application
    ```
    cd app-name
    ```
  - Install the dependencies
    ```
    npm install
    ```
  - Run in development mode
    ```
    npm start
    ```
  - To make a build to serve locally
    ```
    npm run build
    ```
